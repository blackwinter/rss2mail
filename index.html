    rss2mail - Send RSS feeds as e-mail¶ ↑

    VERSION¶ ↑

    This documentation refers to rss2mail version 0.4.4

    DESCRIPTION¶ ↑

    Sends new entries for configured RSS feeds as e-mail to any number of recipients. See example/ directory for a sample configuration file.
